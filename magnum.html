MAGNUM OPUS

NINA DEMIRJIAN

Magnum Opus is a two-player anti-deck building game for iOS. The game was developed in Swift and was originally created for CSCI-UA 400 iOS, an elective computer science class at NYU.

I was part of a two-person team for this project. I created the UI and menu screen and made all of the original artwork for the game. My teammate, Elliot Winch, designed and developed the gameplay.

The theming of the game is influenced by tarot cards, with each card corresponding to a classic tarot card, such as “The Empress”, “Justice”, “Swords”, “The Devil” and “The Magician”.
